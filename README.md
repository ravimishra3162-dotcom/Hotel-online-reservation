# Hotel Online Reservation Analysis

## Overview
This project analyzes hotel online reservation data to uncover patterns, trends, and insights that can help hotels optimize bookings, understand customer preferences, and increase revenue. The analysis includes city wise reservations, top booking websites, revenue trends and more.

## Project Structure
- `data/` - Raw dataset and data documentation.
- `notebooks/` - Jupyter notebooks for data cleaning, exploration, visualization, and insights.
- `scripts/` - Reusable Python scripts for cleaning, analysis, and visualization.
- `reports/` - Exported figures and PDF reports.

## Dataset
The dataset contains the following columns:
- `Reservation_id`: Unique identifier for each reservation
- `Reservation_Date`: Date the reservation was made
- `Hotel_Name`: Name of the hotel
- `City`: City where the hotel is located
- `Booking_Website`: Platform used to book the hotel
- `CheckIn_Date`: Check-in date
- `CheckOut_Date`: Check-out date
- `Rooms`: Number of rooms booked
- `Guests`: Number of guests
- `Price_USD`: Total reservation price in USD
